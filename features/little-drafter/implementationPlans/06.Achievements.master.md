## 1. Feature Overview
- **Name**: Achievements
- **Goal**: Track completed drawings and award stars/badges via a projection.
- **Complexity**: Simple
- **Dependencies**: Draft Domain (emits `DraftCompleted`).

---

## 2. Implementation Phases

### Phase 1: Achievement Projection
- Implement `AchievementProjection` listening for `DraftCompleted`.
- Update the `Achievements` collection in MongoDB: `upsert` user record and `$inc` the `completedDrawings` count.

### Phase 2: Reward API (Optional/Simple)
- Expose a simple query to fetch user achievement counts for display in the UI.
