## 2. Sequence Diagram (Happy Path)

```mermaid
sequenceDiagram
    participant U as User
    participant UI as DrawingSession
    participant CS as CatalogService
    participant DD as DraftDecider
    participant ES as Event Store
    participant AP as AchievementProjection

    U->>UI: Selects Template
    UI->>CS: Fetch SVG Metadata
    CS-->>UI: Template (Steps, Instructions)
    UI->>DD: StartDraft Command
    DD->>ES: Emit DraftStarted
    UI->>UI: Render Step 0 + Pulse + TTS

    U->>UI: Clicks "Next"
    UI->>DD: ProgressStep Command
    DD->>ES: Emit DraftStepProgressed
    UI->>UI: Render Step 1 + Pulse + TTS

    U->>UI: Clicks "Next" (Final)
    UI->>DD: ProgressStep Command
    DD->>ES: Emit DraftCompleted
    ES-->>AP: Trigger Projection
    AP->>AP: Update Star Count
    UI->>UI: Trigger Celebration
```
