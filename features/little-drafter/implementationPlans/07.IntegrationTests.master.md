## 3. Scenario Specifications

### Scenario 1: Full Drawing Lifecycle (Pipeline Test)
- **Given**: A registered "Lion" template in the catalog.
- **When**:
    1. Send `StartDraft("lion")` command via Command Bus.
    2. Send `ProgressStep` commands until completion.
- **Then**:
    - `DraftCompleted` event is found in the Event Store.
    - `AchievementProjection` has updated the user's star count in MongoDB to `1`.

### Scenario 2: Catalog and UI Orchestration
- **Given**: A mocked `CatalogService`.
- **When**: UI requests a template and dispatches a command.
- **Then**: Verify that the command payload matches the template metadata requirements.
