### 00.Scenarios.master.md Findings
- **Key Scenarios**: 11 scenarios covered, including lifecycle (Start/Complete), Interaction (Forward/Undo), Guidance (Pulse/TTS), and Edge Cases (Rapid Click, TTS Failure).
- **Architecture Validation**: Emmett Decider fits the sequential nature perfectly.

### Insights (DevInsights.md & DeepDive.md)
- **Decisions**: 
    - Single session aggregate for "Draft".
    - `DraftCompleted` event for rewards.
    - Manual SVG grouping/layering.
    - Large colorful arrows in a dedicated bottom bar.
    - Blue pulse (`#0000FF`) for guidance.
- **Deep Dive Integration**: 
    - `packages/backend` using ES-Kit and Emmett.
    - `packages/frontend` using Next.js 15 and `framer-motion`.
    - SVG metadata extraction strategy (`data-step`, `data-instruction`).
